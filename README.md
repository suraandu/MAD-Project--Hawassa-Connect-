# MAD-Project--Hawassa-Connect-
App Name= "Hawassa Connect"

Tagline: "Your city’s inventory in your pocket find it locally, buy it personally"

Description: Hawassa-Connect is a "Local Inventory Discovery" app. It allows sellers to list their current stock and prices in a digital storefront. Buyers can search for items by category or location, compare prices, and get the shop's exact location and contact info. The app facilitates the discovery and decision-making online, while the transaction remains a safe, physical exchange at the shop.
